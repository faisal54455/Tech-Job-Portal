## 📌 Overview

A full-stack job portal built using Django that aggregates job listings through web scraping and allows admin-controlled user management.

## 🚀 Features

* 🔍 Job scraping from external websites
* 👤 User registration & login system
* 🛡️ Secure authentication
* 🧑‍💼 Admin dashboard to manage users & jobs
* 📄 Job listing and filtering

## 🛠️ Tech Stack

* Backend: Python, Django
* Frontend: HTML, CSS, Bootstrap
* Database: SQLite (can be extended to PostgreSQL)
* Web Scraping: BeautifulSoup / Requests

## ⚙️ Installation

```bash
git clone https://github.com/your-username/job-portal-django.git
cd job-portal-django
pip install -r requirements.txt
python manage.py runserver
```

## 🔐 Security Features

* Django authentication system
* CSRF protection
* Form validation
* Admin access control
